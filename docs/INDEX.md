## Project Overview

Save a Stray is a pet adoption platform that connects animal shelters with potential adopters using a modern GraphQL API and Apollo Client.

### Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React, Apollo Client, React Router |
| Backend | Node.js, Express, GraphQL |
| Database | MongoDB, Mongoose |
| Auth | Passport.js (Google, Facebook OAuth) |
